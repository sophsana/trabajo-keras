# Trabajo: Biblioteca Keras en Python

Este repositorio contiene un ejemplo sencillo del uso de la biblioteca Keras,
aplicado a una problemática del contexto educativo colombiano.

## Contenido
-`keras_biblioteca.pdf`: Documento escrito de la biblioteca Keras.
- `modelo_keras.py`: Código en Python usando Keras para predecir si un estudiante aprueba o no una asignatura.

## Descripción del problema
Se busca predecir el resultado académico de un estudiante colombiano a partir
de sus horas de estudio semanales y su porcentaje de asistencia a clases.

## Biblioteca utilizada
- Keras (TensorFlow)
